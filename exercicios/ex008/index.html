Principais formatações

Negrito / destaque

Nesta frase temos um termo em negrito usando a tag <B> (não semantico)
Nesta frase temos um termo em negrito usando a tag <strong> (mais semantico)

Itálico / ênfase

Nesta frase temos um termo em Itálico usando a tag <i> (não semantico)
Nesta frase temos um termo em Itálico usando a tag <EM> (semantico)

Texto marcado

Podemos criar tambem um texto marcado usando a tag <mark>

Texto grande e pequeno

aqui esta um texto pequeno usando a tag <small> (para deixar maior, apenas com CSS ou usando <big>, mas esta obsoleto)

Texto deletado

podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado, usando a tag <del>

texto inserido

podemos marcar um texto como inserido par adar uma ênfase e indicar que ele foi adicionado depois com a tag <ins>

Texto sobrescrito

para inserir algo superior/elevado como: x20+3, usando a tag <sup>

Texto Subescrito

para inserior algo abaixo/inferior como: H2O, usando a tag <sub>
